# tgift_prices

> **⚠️ Ранняя Альфа-версия:** Этот проект находится на самой ранней стадии разработки. Текущая версия включает только начальные функции (proof-of-concept). Возможности и структура API могут сильно измениться в будущем.

Простой инструмент на Python, который мы создаем для получения минимальных цен на подарки в Telegram с различных торговых площадок.

В настоящее время `tgift_prices` предоставляет три базовые функции:

- получение минимальной (floor) цены на подарок
- поиск названий подарков по внутреннему списку
- получение списка поддерживаемых подарков

## Быстрый пример

```python
from tgift_prices import get_floor

# Получение текущей минимальной цены (возвращает данные или None)
result = get_floor("Heart Locket")
print(result)
```

Пример ответа:

```python
{
    "Heart Locket": {
        "floor": 12.5,
        "value": "TON",
        "marketplace": "TONNEL"
    }
}
```
