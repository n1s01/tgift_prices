# tgift_prices

> Быстрый доступ к минимальным ценам на подарки в Telegram с разных торговых площадок.

`tgift_prices` — это компактная библиотека Python для трех простых задач:

- получение минимальной (floor) цены на один или несколько подарков
- поиск названий подарков по встроенному каталогу
- получение полного списка доступных подарков

## Быстрый пример

```python
from tgift_prices import get_floor

result = get_floor("Heart Locket")
print(result)
```

Пример ответа:

```python
{
    "Heart Locket": {
        "floor": 12.5,
        "value": "TON",
        "marketplace": "TONNEL"
    }
}
```

## Почему стоит использовать эту библиотеку

- Небольшая площадь покрытия API и предсказуемый формат возвращаемых данных
- Список торговых площадок по умолчанию из коробки
- Встроенная валидация по известному каталогу подарков
- Подходит для запросов как одного подарка, так и пакета подарков
